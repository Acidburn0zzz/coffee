# Coffee

Keep up-to-date with news and weather with Coffee.

## Building and Installing from code

You'll need the following dependencies:

	libglib2.0-dev
	libgtk-3-dev
	libjson-glib-dev
	libsoup2.4-dev
	libwebkit2gtk-4.0-dev
	libgee-0.8-dev
	libgeocode-glib-dev
	libgeoclue-2-dev
	libkeybinder-3.0-dev
	libnotify-dev
